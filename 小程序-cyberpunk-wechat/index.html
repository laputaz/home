云开发

云开发环境创建的时候, 一个账号可以提供两个环境, 每个环境可以设置 id

  1. 云数据库
    json 形式数据库, 每一条数据都是 json 格式的对象 , 与关系型数据库差异如图, 基于 mongoDB
    关系型数据库更适合表之间有复杂的关联关系, 文档型更适合频繁的读写操作, 效率高一些

    包含以下数据类型

    在小程序端操作会有权限控制, 而云函数操作则有完全的读写控制

  1. 云函数
    云函数, 保存在云

  2. 云存储
    云存储能力

  • wx.cloud.downloadFile //下载
