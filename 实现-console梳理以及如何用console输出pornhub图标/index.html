再夸张一点

console 其实还有很多有用的方法

console.warn

警告

console.error

错误

console.count

计数, 相同的 label 数量会相加 (可以用来统计函数调用次数)

console.assert

断言, 断言为 false，则将打印一个错误消息

console.debug

只有在level包含Verbose的时候才显示

console.info

只有在level包含Info的时候才显示

console.dir

打印出该对象的所有属性和属性值.

和普通的log好像没什么区别?

console.group() / console.groupEnd()

分组

console.table

表格得形式展示对象/数组

console.time() / console.timeEnd()

可以用于计时(参数用于区分多个timer, 也用于显示打印的label)
